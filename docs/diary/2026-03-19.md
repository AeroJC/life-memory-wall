# Diary — 2026-03-19

## Summary
Added three new features: "On This Day" memories, PDF/Book Export, and Video moments substory type.

## Q&A Log
**Q:** Implement 3 features — On This Day reminders, PDF/Book Export, and Video moments.
**A:** Implemented all three features across both frontend and backend:

1. **"On This Day" Reminders** — Backend endpoint (`GET /api/on-this-day`) queries memories matching today's month/day from previous years across all user spaces. Frontend component shows a carousel card on the SpaceSelector page with year-ago badges and navigation.

2. **PDF/Book Export** — Frontend-only jsPDF implementation. Users click a Download button on the Memory Detail (moments) page to export a single memory as a styled PDF. Generates cover page (cover photo background + dark overlay + title/date/location), story page, additional cover photos, tags, then each moment on its own page (number label, title, date, divider, photos, text, video/audio notes), and a back cover.

3. **Video Moments** — New `video` substory type. Added `videoUrl` field to Prisma schema and SubStory type. Frontend supports video upload to Cloudinary (up to 100MB, MP4/WebM/MOV/AVI/MKV), inline HTML5 video player in detail view, video layout option in both add and edit forms.
